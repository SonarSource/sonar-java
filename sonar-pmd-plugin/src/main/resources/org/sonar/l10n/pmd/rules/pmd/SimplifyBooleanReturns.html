Avoid unnecessary if..then..else statements when returning a boolean. Example : 
public class Foo {
  private int bar =2;
  public boolean isBarEqualsTo(int x) {
    // this bit of code
    if (bar == x) {
     return true;
    } else {
     return false;
    }
    // can be replaced with a simple
    // return bar == x;
  }
}

This rule is deprecated, use {rule:squid:S1126} instead.
