A couple Collection methods can be called with arguments of an incorrect type, but doing so is pointless and likely the result of using the wrong argument.

Noncompliant Code Example

List list = new ArrayList();
Integer integer = Integer.valueOf(1);

if (list.contains(integer)) {  // Noncompliant. Always false.
  list.remove(integer); // Noncompliant. list.add(iger) doesn't compile, so this will always return false
}
