Declaring multiple variable on one line is difficult to read.

Noncompliant Code Example

class MyClass {

  private int fieldOne, fieldTwo;

  public void method(){
    int varOne, varTwo;
  }
}

Compliant Solution

class MyClass {

  private int fieldOne;
  private int fieldTwo;

  public void method(){
    int varOne;
    int varTwo;
  }
}
