This rule allows you to track the usage of the @SuppressWarnings mechanism. You can use one of two modes:

  1. BLACKLIST - listOfWarnings stores not allowed suppressions, this is a default mode

    Noncompliant Code Example

    In case "listOfWarnings=unused"

    @SuppressWarnings("unused")

    BLACKLIST Compliant Solution

    In case "listOfWarnings=all"

    @SuppressWarnings("unchecked")
@SuppressWarnings("unused")
  2. WHITELIST - listOfWarnings stores allowed suppressions

    Noncompliant Code Example

    In case "listOfWarnings=serial"

    @SuppressWarnings("unused")

    Compliant Solution

    In case "listOfWarnings=serial"

    @SuppressWarnings("serial")
