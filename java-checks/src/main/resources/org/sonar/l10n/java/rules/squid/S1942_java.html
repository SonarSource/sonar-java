Java's import mechanism allows the use of simple class names. Therefore, using a class' fully qualified name in a file that imports the class is redundant and confusing.

Noncompliant Code Example

import java.util.*;
import java.sql.Timestamp;

//...

java.util.List myList <String> =   // Noncompliant; covered by java.util.* import
    new java.util.ArrayList <String> (); // Noncompliant; covered by java.util.* import

java.sql.Timestamp tStamp; // Noncompliant

Compliant Solution

import java.util.*;
import java.sql.Timestamp;

//...

List myList <String> =  new ArrayList <String> ();
Timestamp tStamp;
