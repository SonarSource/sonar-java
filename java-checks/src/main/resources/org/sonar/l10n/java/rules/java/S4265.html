FixedSizeList from the Commons library, the list returned from Arrays.asList and some other offer methods (as required by the List interface they implement), but neither truly supports their use. All this types have fixed lengths and will throw errors if a method is called.

Noncompliant Code Example

List<String> strings = Arrays.asList("Hello");
strings.add("world");  // Noncompliant

Compliant Solution

List<String> strings = Arrays.asList("Hello", "world");
